    YouTube API Setup

    1. Go to console.cloud.google.com
    2. Create a new project
    3. Enable YouTube Data API v3
    4. Create OAuth 2.0 credentials (Desktop app)
    5. Download the JSON file
    6. Upload it in Settings → YouTube Upload → Upload JSON
